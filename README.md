# Cardano Community Activities

Repository containing tools and applications for managing Cardano community activities and offline operations for end-users.

## Overview

This project provides cross-platform applications (PowerShell, C#, Python, Web) to:
- Manage Cardano wallets and private keys securely
- Handle community activity coordination and tracking
- Recover wallets from recovery phrases
- Verify and monitor blockchain transactions
- Work offline with local data synchronization

## Project Structure

```
cardano-community-activities/
├── misc/               # PowerShell GUI tools for community management
├── lib/                # Shared PowerShell utility libraries
├── local app/          # Windows applications (WinUI, PowerShell)
├── web app/            # Backend Node.js + Frontend Vue/Vite
├── electron-app/       # Cross-platform Electron app with Python backend
└── python/             # Standalone Python scripts and utilities
```

## Main Tools

1. **Community Activity Manager** - PowerShell GUI for managing wallets and activities
2. **WinUI Applications** - Modern Windows applications with native UI
3. **Web Dashboard** - Web-based interface for activity monitoring
4. **Electron App** - Cross-platform application with Python backend
